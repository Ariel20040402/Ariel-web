# Ariel's Personal Website - 个人作品集网站

## 项目介绍
这是一个充满创意和个性化的个人作品集网站，以「Ariel的房间」为核心视觉概念，通过交互式热点探索的方式展示个人的语言学习、项目经历、旅行记忆、手工作品、美食分享等多个维度的内容。网站采用温暖柔和的奶油色系设计风格，搭配流畅的动画效果，打造沉浸式的浏览体验。

## 核心功能特点

### 🌟 视觉与交互
- **沉浸式加载动画**：带有进度条、弹跳表情和脉冲标题的加载页面，提升用户等待体验
- **交互式房间探索**：首页以房间背景为基础，点击不同物品热点（笔记本、蛋糕、蜡烛等）进入对应内容页面
- **流畅的过渡动画**：所有按钮、卡片、热点都配有精心设计的hover和transition效果
- **响应式设计**：适配移动端和桌面端，保证不同设备下的浏览体验

### 📋 内容模块
1. **语言学习页面** - 展示法语(TCF C1)、英语(雅思7.5)和中文能力
2. **项目经历页面** - 时间轴形式展示产品经理实习项目经验
3. **探索世界页面** - 旅行照片展示，支持图片切换浏览
4. **小红书笔记页面** - 个人笔记卡片展示与添加功能
5. **美食分享页面** - 美食内容展示
6. **手工作品页面** - 手工作品展示
7. **碎碎念页面** - 个人想法与留言板功能

### 🎵 额外功能
- 背景音乐播放/暂停控制
- 图片容错处理（加载失败时优雅降级）
- 卡片悬停效果与交互反馈
- 表单提交功能（留言板、添加笔记）

## 技术栈
- HTML5 (语义化标签、多媒体支持)
- CSS3 (自定义属性、动画、网格布局、弹性布局)
- JavaScript (交互逻辑、DOM操作、事件处理)
- Font Awesome 图标库
- 响应式设计 (Media Queries)

## 色彩体系
网站采用温暖柔和的奶油色系为主调，主要颜色变量：
- `--cream-1: #F7EFE3` (主背景色)
- `--cream-2: #FAF4EA` (页面背景色)
- `--cream-3: #EFE1CF` (强调色)
- `--soft-pink: #FFE4E1` (柔和粉色)
- `--wood-light: #D4B896` (木质浅棕色)
- `--text-soft: #6B5B4F` (柔和文本色)
- `--text-dark: #4A3F35` (深色文本)

## 使用说明
1. 将所有文件放在同一目录下，确保图片和音乐文件路径正确
2. 直接打开 `index.html` 文件即可运行
3. 加载完成后点击 "ENTER" 按钮进入主页面
4. 在房间页面点击不同物品热点探索对应内容
5. 右上角音乐图标可控制背景音乐播放/暂停
6. 各页面左上角的「回到房间」按钮可返回首页

## 自定义指南
### 修改内容
- 替换图片：替换 `images/` 目录下的图片文件，保持文件名一致
- 修改文字内容：直接在HTML中对应位置修改文本
- 更新项目经历：编辑 `projectsPage` 中的项目时间轴内容
- 修改语言能力：编辑 `languagePage` 中的语言卡片信息

### 修改样式
- 调整颜色：修改CSS中 `:root` 里的颜色变量
- 调整动画：修改CSS中的 `@keyframes` 动画定义
- 调整布局：修改网格布局(`grid`)或弹性布局(`flex`)相关属性

### 添加新页面
1. 复制现有页面模板，修改ID和内容
2. 在房间页面添加对应的热点按钮
3. 编写切换页面的JavaScript函数

## 适配说明
- 移动端：适配768px以下屏幕，优化布局和交互
- 桌面端：完整展示所有视觉效果和布局
- 图片加载失败时会自动隐藏，保证页面美观
